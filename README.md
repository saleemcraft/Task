# Laravel Ecommerce Project

## Features
- Product CRUD
- Image Upload
- Pagination
- Edit & Delete

## Technologies
- Laravel
- MySQL
- Bootstrap

## Setup
1. Clone repo
2. Run `composer install`
3. Copy `.env.example` to `.env`
4. Run `php artisan key:generate`
5. Run `php artisan migrate`
6. Run `php artisan serve`
